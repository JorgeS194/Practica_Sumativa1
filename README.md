# Practica_Sumativa1
Ejecución de código php básico y elemental

🧮 Práctica Sumativa #1 – Introducción a PHP
Proyecto desarrollado como parte de la asignatura Desarrollo de Software VII en la Universidad Tecnológica de Panamá.

📌 Descripción
Este proyecto consiste en la implementación de tres scripts básicos en PHP que realizan operaciones aritméticas fundamentales:
Suma
Resta
Multiplicación

El objetivo es reforzar los conceptos iniciales de PHP, como el uso de variables, operadores y la integración con HTML para mostrar resultados en pantalla.

📂 Estructura del proyecto
PRACTICA_SUMATIVA1/
│
├── suma.php
├── resta.php
└── multiplicacion.php

⚙️ Funcionamiento
Cada script sigue la misma lógica:
1. Se definen dos variables numéricas ($num1 y $num2)
2. Se realiza la operación correspondiente
3. Se muestra el resultado en pantalla utilizando HTML y echo

🧾 Scripts incluidos
➕ suma.php: Realiza la suma de dos números.
➖ resta.php: Realiza la resta de dos números.
✖️ multiplicacion.php: Realiza la multiplicación de dos números.

🚀 Cómo ejecutar el proyecto
Instalar un servidor local como:
XAMPP
WAMP
Laragon
Colocar la carpeta del proyecto en:
htdocs (XAMPP)
www (WAMP)

Ejecutar en el navegador:
http://localhost/PRACTICA_SUMATIVA1/suma.php
http://localhost/PRACTICA_SUMATIVA1/resta.php
http://localhost/PRACTICA_SUMATIVA1/multiplicacion.php


🛠️ Tecnologías utilizadas
PHP
HTML

🎯 Objetivo del proyecto
Comprender el uso de variables en PHP
Aplicar operadores aritméticos básicos
Integrar PHP con HTML
Visualizar resultados dinámicamente en el navegador
